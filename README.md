# 通义万相2.7 视频生成器

基于阿里云通义万相AI模型的两阶段视频生成系统,实现从文本描述到视频生成的完整流程。

## ✨ 功能特性

- 🎨 **文生图**: 使用wan2.7-image-pro模型生成高质量图片(支持1K/2K/4K)
- 🎬 **图生视频**: 使用wan2.7-i2v模型将图片转换为流畅视频（最新旗舰模型）
- 👤 **人物一致性**: 支持上传参考图保障人物特征一致性
- ⚡ **一键生成**: 自动完成文生图→图生视频完整流程
- 📊 **实时监控**: 任务状态实时轮询,进度可视化
- 💾 **本地存储**: 自动下载生成的图片和视频到本地
- 🎯 **灵活控制**: 支持分步生成,先生成满意图片再生成视频

## 🏗️ 技术架构

```
前端: HTML5 + CSS3 + JavaScript (原生)
后端: Python Flask + requests
AI模型: 
  - 文生图: wan2.7-image-pro (同步调用)
  - 图生视频: wan2.7-i2v (异步调用，最新旗舰模型)
```

## 📋 前置要求

- Python 3.7+
- 阿里云百炼API Key (北京地域)
  - 文生图API Key
  - 图生视频API Key

### 获取API Key

1. 访问阿里云百炼控制台: https://bailian.console.aliyun.com/cn-beijing?tab=api
2. 登录阿里云账号
3. 进入"API-KEY管理"页面
4. 创建API Key并复制保存
5. **重要**: 确保API Key属于**北京地域(cn-beijing)**

## 📖 使用说明

### 方式一: 分步生成(推荐)

#### 步骤1: 生成图片

1. **(可选)** 上传人物参考图
   - 上传单张参考图片(保障人物一致性)
   - 支持常见图片格式(JPG, PNG等)

2. 填写图片描述
   - 详细描述你想要的图片内容
   - 包括人物、场景、光影、风格等
   - 最多5000字符

3. **(可选)** 填写反向提示词
   - 描述不想在画面中出现的内容

4. 选择图片尺寸
   - 1K (1024×1024)
   - 2K (2048×2048) - 推荐
   - 4K (4096×4096)

5. 点击"🖼️ 生成图片"按钮

6. 等待生成完成,查看生成的图片

#### 步骤2: 生成视频

1. 确认已生成满意的图片

2. 填写视频描述
   - 描述视频中的动作和场景变化
   - 最多1500字符

3. 选择视频参数
   - 分辨率: 720P(推荐) / 1080P
   - 时长: 2-15秒

4. 点击"🎥 生成视频"按钮

5. 等待任务完成(通常1-5分钟)
   - 进度条实时显示状态
   - 完成后自动展示视频播放器

6. 播放或下载生成的视频

### 方式二: 一键生成

1. 同时填写图片描述和视频描述
2. (可选)上传参考图和设置参数
3. 点击"🚀 一键生成"按钮
4. 系统自动完成完整流程
5. 等待最终视频生成完成

## ⚙️ API接口说明

### 后端API

- `POST /api/generate-image` - 生成图片
- `POST /api/generate-video` - 创建视频生成任务
- `POST /api/upload-image` - 上传参考图片
- `GET /api/task/<task_id>` - 查询任务状态
- `POST /api/full-pipeline` - 一键生成完整流程
- `GET /images/<filename>` - 访问生成的图片
- `GET /videos/<filename>` - 访问生成的视频

## 💡 使用技巧

### 图片描述提示词技巧

- **具体描述**: "一位身穿蓝色风衣的年轻侦探,站在雨夜的纽约街头"
- **光影效果**: "霓虹灯闪烁,电影级光影效果,赛博朋克风格"
- **构图视角**: "中景镜头,人物位于画面右侧三分之一处"
- **情绪氛围**: "忧郁的眼神望向远方,孤独而神秘的氛围"

### 视频描述提示词技巧

- **动作描述**: "侦探缓缓转身,望向远方的霓虹灯"
- **场景变化**: "雨水从风衣上滑落,远处传来警笛声"
- **镜头运动**: "镜头缓慢推近,聚焦人物面部表情"
- **时间节奏**: "慢动作效果,雨滴在空中凝固"

### 人物一致性保障

1. **单张参考图**(简化方案)
   - 选择最清晰的照片作为参考
   - 可以是正面、侧面或其他角度
   - 系统会自动学习人物特征

3. **描述中强调**
   - 在prompt中说明"保持人物特征一致"
   - 描述人物的关键特征(发型、服装等)

## ⚠️ 注意事项

### 费用说明

- **文生图**: 按生成图片数量计费
  - wan2.7-image-pro: 根据分辨率不同计费
  - 4K分辨率费用最高
  
- **图生视频**: 按视频时长和分辨率计费
  - wan2.7-i2v: 根据分辨率和时长计费
  - 1080P费用高于720P

**请在调用前查看百炼控制台最新价格**

### 限制说明

- **图片生成时间**: 通常5-30秒
- **视频生成时间**: 通常1-5分钟
- **图片尺寸**: 最大4K(4096×4096)
- **视频时长**: 2-15秒 (wan2.7-i2v)
- **Prompt长度**: 
  - 文生图: 最多5000字符
  - 图生视频: 最多1500字符
- **参考图大小**: 单张不超过20MB
- **task_id有效期**: 24小时

### 最佳实践

1. **先图后视频**: 建议先生成满意的图片,再生成视频
2. **控制成本**: 先用低分辨率测试效果,满意后再用高分辨率
3. **保存素材**: 及时下载保存生成的图片和视频
4. **参考图质量**: 使用高质量、清晰的参考图效果更好
5. **详细描述**: 提示词越详细,生成效果越符合预期

## 🔧 常见问题

### Q: API调用失败怎么办?

A: 检查以下几点:
1. API Key是否正确且属于北京地域
2. 网络连接是否正常
3. 账户余额是否充足
4. 查看控制台输出的错误信息

### Q: 视频生成时间很长?

A: 正常现象:
- 图生视频是异步任务,通常需要1-5分钟
- 高峰期可能更长
- 请耐心等待,进度条会实时更新

### Q: 人物一致性不好?

A: 尝试以下方法:
1. 上传更清晰、更高质量的参考图
2. 在prompt中详细描述人物特征
3. 使用更高分辨率生成图片

### Q: 生成的视频不满意?

A: 可以:
1. 重新生成图片(调整prompt)
2. 使用同一张图片生成多次(随机种子不同)
3. 修改视频描述prompt

### Q: 如何修改端口?

A: 编辑 `.env` 文件:
```env
FLASK_PORT=8080
```
