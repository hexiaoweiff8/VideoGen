## 1. 环境准备与启动

### 1.1 系统要求

- Python 3.8 及以上版本
- 可联网环境（需访问阿里云百炼 API）
- 阿里云百炼账号（需开通文生图和视频生成服务）

### 1.2 安装依赖

在项目根目录打开终端，执行：

```bash
pip install -r requirements.txt
```

所需依赖（`requirements.txt`）：

| 包名 | 版本 | 说明 |
|------|------|------|
| Flask | 3.0.0 | Web 框架 |
| requests | 2.31.0 | HTTP 请求库 |
| python-dotenv | 1.0.0 | 环境变量读取 |

### 1.3 配置 API Key

在项目根目录找到 `.env` 文件（或复制 `.env.example` 重命名为 `.env`），填入你的 API Key：

```ini
# API Keys（在阿里云百炼控制台获取）
IMAGE_API_KEY=sk-你的文生图API Key
VIDEO_API_KEY=sk-你的视频生成API Key

# Flask 服务配置（一般不需要修改）
FLASK_HOST=127.0.0.1
FLASK_PORT=5000
FLASK_DEBUG=True
```

> **如何获取 API Key：**
> 1. 登录 [阿里云百炼控制台](https://bailian.console.aliyun.com/)
> 2. 点击右上角头像 → **API-KEY 管理**
> 3. 创建新的 API Key，复制后填入 `.env`

### 1.4 启动应用

```bash
python app.py
```

启动成功后终端显示：

```
 * Running on http://127.0.0.1:5000
 * Debug mode: on
```

### 1.5 访问页面

打开浏览器，访问：

```
http://127.0.0.1:5000
```

│     🎨 文生图        │   🎬 图文生视频       │

## 3. 文生图功能（左侧区域）

### 3.1 完整操作流程

```
第①步  上传参考图（可选）
    ↓
第②步  填写图片描述
    ↓
第③步  选择图片尺寸
    ↓
第④步  点击「生成图片」
    ↓
第⑤步  等待生成（通常 10~30 秒）
    ↓
第⑥步  查看 / 下载图片
```

### 3.2 人物参考图（可选）

用于保障生成图片中**人物形象的一致性**（如固定角色外观）。

- 点击参考图上传框，选择本地图片
- 支持格式：JPG、PNG、WEBP 等常见格式
- 文件大小：不超过 **20 MB**
- **不上传则忽略此步**，直接填写描述即可

### 3.3 图片描述（必填）

- **图片描述文本**（必填）：详细描述想要生成的画面内容
  - 最多 **5000 字符**
  - 建议包含：主体、风格、光线、色调、构图等
  - 示例：`一位穿着汉服的年轻女性，站在樱花树下，春日午后，柔和光线，水墨画风格`

- **反向提示词**（可选）：描述不希望出现在画面中的内容
  - 示例：`模糊、变形、低质量、多余的肢体`

### 3.4 图片尺寸选择

| 选项 | 实际分辨率 | 适用场景 |
|------|-----------|----------|
| 1K | 1024 × 1024 | 快速预览、低消耗 |
| **2K（推荐）** | 2048 × 2048 | 日常使用，画质与速度均衡 |
| 4K | 4096 × 4096 | 高质量输出，生成较慢 |

### 3.5 点击「生成图片」

- 点击后页面底部状态栏显示进度
- 图片生成完成后自动展示在左侧结果区
- 可点击 **💾 下载** 保存到本地

## 4. 图文生视频功能（右侧区域）

### 4.1 完整操作流程

```
第①步  提供首帧图片（上传本地图片 或 输入图片URL）
    ↓
第②步  填写视频描述
    ↓
第③步  设置分辨率和时长
    ↓
第④步  点击「生成视频」
    ↓
第⑤步  等待生成（通常 2~5 分钟）
    ↓
第⑥步  查看 / 下载视频
```

### 4.2 提供首帧图片（必填）

右侧区域**独立**管理图片来源，有两种方式：

#### 方式一：上传本地图片（推荐）

1. 点击 **「📁 上传本地图片」** 按钮
2. 从本地选择图片文件（JPG/PNG/WEBP，不超过 20 MB）
3. 图片会自动上传到阿里云临时存储，页面显示预览
4. 上传成功后提示：`✅ 上传成功！有效期 48 小时`
5. 此时即可直接点击「生成视频」

> 💡 本地图片经过阿里云 OSS 临时存储中转，API 可直接访问，无需手动处理图片外链问题。

#### 方式二：输入公网图片 URL

1. 在输入框中粘贴图片的**公网可访问地址**（以 `https://` 开头）
2. 确保该 URL 可在浏览器中直接打开（不能是本地路径或需要登录的地址）
3. 输入 URL 后会自动清除之前上传的图片

> ⚠️ **注意**：两种方式互斥。输入 URL 后会清除上传状态；重新上传图片后会清除 URL 输入框。

### 4.3 视频描述（必填）

描述视频中的**动作和场景变化**：

- 最多 **1500 字符**
- 建议描述：主体动作、镜头运动、场景氛围
- 示例：`女性缓缓转身，长发随风飘动，镜头轻微推进，樱花花瓣飘落`

### 4.4 视频参数设置

| 参数 | 选项 | 说明 |
|------|------|------|
| 分辨率 | **720P（推荐）** / 1080P | 1080P 生成更慢，消耗更多配额 |
| 时长 | 2 ~ 15 秒，默认 5 秒 | 时长越长生成越慢 |

### 4.5 点击「生成视频」

- 点击后开始创建异步任务，底部状态栏实时显示进度
- 系统每 **5 秒** 自动查询一次任务状态
- 任务完成后视频自动展示在右侧结果区，支持在线预览
- 可点击 **💾 下载** 保存到本地

## 5. 参数设置详解

### 5.1 提示词写作建议

**文生图提示词结构**（建议顺序）：

```
[主体描述] + [风格/艺术形式] + [环境/场景] + [光线/色调] + [构图/视角]
```

示例：
```
身着红色旗袍的中年女性，写实摄影风格，北京胡同背景，
黄昏暖光，半身特写，浅景深
```

**视频提示词结构**（建议顺序）：

```
[主体动作] + [镜头运动] + [场景变化] + [氛围情绪]
```

示例：
```
女性缓缓抬起头微笑，镜头从远景慢慢推近，
背景樱花轻轻飘落，温柔宁静的氛围
```

### 5.2 图片尺寸与生成时间对照

| 尺寸 | 参考生成时间 | 推荐场景 |
|------|------------|---------|
| 1K | 10~20 秒 | 快速测试提示词效果 |
| 2K | 15~30 秒 | 常规使用（默认推荐） |
| 4K | 30~60 秒 | 需要高清大图输出时 |

### 5.3 视频时长与生成时间对照

| 视频时长 | 720P 参考时间 | 1080P 参考时间 |
|---------|-------------|--------------|
| 2~5 秒 | 1~3 分钟 | 2~5 分钟 |
| 6~10 秒 | 2~5 分钟 | 4~8 分钟 |
| 11~15 秒 | 4~8 分钟 | 7~12 分钟 |

> ⏱️ 生成时间受阿里云服务器负载影响，高峰期可能更长，系统最长等待 **10 分钟**。

## 7. 注意事项与常见问题

### 7.1 API 使用须知

- 文生图和视频生成使用**各自独立的 API Key**，分别在 `.env` 中配置
- 调用会消耗阿里云百炼账号的**配额/余额**，请关注账单
- 阿里云百炼新用户通常有**免费试用额度**，具体以官网为准

### 7.2 本地图片上传注意事项

- 上传的图片经阿里云 OSS 临时存储，**有效期 48 小时**
- 超过有效期后，以该图片生成的视频任务将无法重新提交
- 建议同一张图片当天完成视频生成

### 7.3 常见问题

#### ❓ 点击「生成图片」后一直转圈，没有结果

- 检查 `.env` 中 `IMAGE_API_KEY` 是否正确
- 检查网络是否可访问 `dashscope.aliyuncs.com`
- 查看终端日志，确认是否有错误信息

#### ❓ 视频生成提示"创建任务失败"

- 检查 `.env` 中 `VIDEO_API_KEY` 是否正确
- 确认所选的图片 URL/上传图片是有效的公网可访问地址
- 本地路径（如 `C:/Users/...`）无法使用，请使用上传功能

#### ❓ 上传本地图片后提示"上传失败"

- 检查图片文件大小是否超过 20 MB
- 检查网络连接是否正常
- 确认 `VIDEO_API_KEY` 有效（上传使用视频 API Key 的 OSS 权限）

#### ❓ 视频生成成功但页面没有显示视频

- 等待几秒后刷新页面（视频文件下载需要时间）
- 如果下载失败，系统会自动切换为阿里云原始视频 URL 播放

#### ❓ 生成任务"超时"

- 任务超时限制为 **10 分钟**
- 高峰期服务器繁忙，可稍后重试
- 可以尝试缩短视频时长或降低分辨率

#### ❓ 端口 5000 被占用无法启动

修改 `.env` 文件中的端口号：

```ini
FLASK_PORT=8080
```

重启后访问 `http://127.0.0.1:8080`
